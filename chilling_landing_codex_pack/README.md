# Chilling Barber Shop Landing Page

This is a Vite application, so the source `index.html` cannot be opened directly with a `file://` URL.

## Open the website on Windows

Double-click `START_WEBSITE.cmd`. It starts the local Vite server and opens:

```txt
http://127.0.0.1:4173/
```

Keep the server window running while viewing the website. Close that window when finished.

## Development commands

```powershell
npm install
npm run dev
npm run typecheck
npm run build
```
